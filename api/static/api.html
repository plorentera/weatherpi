API local de Meteo Station

Esta seccion es util si vas a integrar la estacion con otro sistema, dar soporte o desarrollar encima de la API.

Abrir Swagger Abrir guia extendida OpenAPI JSON
Cuándo usarla
  • Integrar con otro sistema local
  • Automatizar tareas
  • Soporte y diagnostico
  • Desarrollo y pruebas
Endpoints mas utiles
Metodo Ruta Uso
GET/api/statusEstado general de la estacion
GET/api/configConfig efectiva y local
PUT/api/configGuardar configuracion local
GET/api/system/accessHost, puerto y URLs utiles
GET/api/system/workersEstado de workers
GET/api/telemetry/statusEstado de envio y outbox
POST/api/telemetry/testEncolar una prueba de envio
GET/api/update/statusEstado de updates
GET/api/remote-config/statusEstado de configuracion remota
Flujo recomendado
  1. Configura la estacion desde Configuracion.
  2. Comprueba la cola de envio en Outbox si algo no llega.
  3. Usa esta API solo cuando necesites integrar, automatizar o diagnosticar.
